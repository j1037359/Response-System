 과목형태에 따라 출석화면이 달라집니다. 과목형태는 e-Class(이클래스)와 e-Learning(이러닝)이 있습니다. * e-Class(이클래스) : 오프라인 강의를 지원합니다.* e-Learning(이러닝) : 온라인으로 진행되는 강의(인터넷 수업)를 지원합니다.[스마트출석]스마트출석은 e-Class(이클래스) 형태의 과목에서 이용가능합니다. 오프라인 강의실에서 수강생이 자신의 스마트 기기를 이용해 자율적으로 출석을 하는 새로운 방식의 출석형태입니다.단, 인터넷 익스플로러 10버전 이상에서만 원활히 이용가능하므로 10버전 이하의 사용자는 인터넷 익스플로러를 업그레이드 해주세요.[스마트출석의 흐름]1. 교수님이 출석시작 버튼을 클릭해 출석을 시작합니다. 2. 출석시작 후 나타나는 인증번호를 수강생에게 알려줍니다.3. 수강생은 자신의 스마트 기기로 접속해 자율적으로 출석을 체크합니다. 이때 인증번호를 정확히 입력해야 출석이 인정됩니다.3. 교수님은 시스템 내에 출석이 체크된 수강생의 정보와 실제 강의실에 있는 수강생을 확인합니다.4. 강의실 내 모든 수강생이 출석체크 했다면 교수님이 출석종료 버튼을 클릭해 출석을 종료합니다.5. 출석정보가 저장됩니다.[스마트출석 등록]1. '출석' 메뉴로 이동한 후 '목록' 탭에서 '스마트출석 등록' 버튼을 클릭합니다. 스마트출석 등록화면으로 이동합니다.2. 스마트출석 등록화면이 표시됩니다.먼저, 날짜와 과목시간을 설정하세요. 날짜 우측의 칸을 클릭하면 달력이 나타납니다. 달력에서 출석일자를 선택하세요.과목시간은 교수님의 구분을 위한 값으로 해당 과목시간으로 설정하세요.날짜와 과목시간을 설정했다면 수강생에게 출석시작을 알린 후 화면 중앙의 '출석시작' 버튼을 클릭하세요. 출석체크가 시작됩니다.3. 출석체크가 시작되면 시간이 흐르고, 인증번호와 '출석종료' 버튼이 나타납니다.교수님은 수강생에게 인증번호를 알려주시면 됩니다.수강생은 자신의 스마트 기기를 이용해 출석화면으로 이동합니다. 수강생은 아래와 같은 화면이 나타나게 됩니다.교수님이 알려주신 인증번호를 정확히 입력한 후 '출석' 버튼을 클릭하면 출석이 인정됩니다.4. 수강생이 출석을 완료하면 교수님의 화면 하단에 해당 수강생의 사진과 이름이 표시됩니다.강의실 내에 해당 수강생이 있는지를 확인해보신 후 강의실 내 모든 수강생이 출석을 완료했다면 '출석종료' 버튼을 클릭해 출석을 종료하세요.'출석취소' 버튼을 클릭하면 출석이 취소됩니다.5. 출석이 종료되면 스마트출석 조회화면으로 이동됩니다.[스마트출석 조회]1.  '출석' 메뉴로 이동한 후 '목록' 탭에서 출석내용을 조회할 출석일을 클릭하세요. 스마트출석 조회화면으로 이동합니다.2. 해당일자의 스마트출석이 조회됩니다. 수강생이 출석한 날짜와 IP 정보가 표시됩니다.수강생의 이름을 클릭하면 수강생의 정보가 레이어 팝업으로 나타납니다.출석하지 않은 수강생을 강제로 출석상태로 변경하려면 '출석' 버튼을 클릭하세요.해당 수강생이 출석상태로 변경되고 붉은색으로 출석허용된 날짜와 IP정보가 표시됩니다.출석하지 않은 수강생을 강제로 지각상태로 변경하려면 '지각' 버튼을 클릭하세요.해당 수강생이 지각상태로 변경되고 붉은색으로 지각허용된 날짜와 IP정보가 표시됩니다.3. 수강생 목록을 정렬하려면 정렬을 설정하세요.정렬의 각 버튼은 이름, 학번, 출결 순서로 정렬되는 것을 나타냅니다.정렬 순서를 바꾸려면 해당 항목을 클릭한 후 원하는 위치로 이동(Drag & Drop)시킵니다.앞쪽(왼쪽)에 위치할수록 정렬 순서에 더 영향을 미칩니다. 항목을 더블클릭하면 오름차순 또는 내림차순으로 변경됩니다.정렬 순서와 정렬 방식(오름차순, 내림차순)을 변경하신 뒤 'Search' 버튼을 클릭하세요. 정렬 조건에 맞게 수강생이 조회됩니다.[스마트출석 재시작과 출석추가]스마트출석에는 재시작 기능과 출석추가 기능이 있습니다. 각각의 예를 들어 안내를 진행하겠습니다.Ex 1) 출석체크를 마치고 강의를 진행하려던 찰나에 학생무리가 들어와서 해당 학생들까지 출석으로 받아줄 경우 -> '재시작' 기능 이용1. 스마트출석 조회화면에서 이전에 진행했던 1차 출석 하단의 'R(재시작)' 버튼을 클릭합니다.2. 해당 차수의 출석을 다시 받을 수 있습니다. 기존에 출석한 수강생의 사진과 이름이 하단에 나타납니다.기존 방식과 마찬가지로 '출석시작' 버튼을 클릭하면 다시 출석체크를 받습니다. 뒤늦게 들어온 수강생이 자신의 스마트 기기를 이용해 출석을 합니다.강의실 내 모든 수강생의 출석이 완료되면 '출석종료' 버튼을 클릭합니다. 1차 출석에 뒤늦게 들어온 수강생의 출석까지 반영되어 저장됩니다.3. 재시작 기능은 이전에 진행한 출석의 차수에 대해서만 사용이 가능합니다.만약, 1차 출석을 진행하고, 2차 출석을 진행한 상태라면 2차 출석만 재시작이 가능합니다.2차 출석을 삭제하면 다시 1차 출석이 재시작 가능한 상태가 됩니다.Ex 2) 강의 진행 중에 잠시 쉬는 시간을 갖고 다시 강의를 진행하려 하니 도망간 학생이 있을 경우 -> '출석추가' 기능 이용1. 스마트출석 조회화면에서 '출석추가' 버튼을 클릭합니다. 출석을 추가할 경우 새로운 차수로 출석이 추가됩니다.2. 기존 방식과 마찬가지로 '출석시작' 버튼을 클릭해 다시 출석체크를 받습니다. 1차로 출석체크를 했던 수강생도 다시 출석체크를 해야합니다.강의실 내 모든 수강생의 출석이 완료되면 '출석종료' 버튼을 클릭합니다. 출석이 종료되면 스마트출석 조회화면으로 이동됩니다.3. 스마트출석 조회화면에 추가된 2차 출석이 표시됩니다. 'X(삭제)' 버튼을 클릭하면 해당 차수의 출석정보가 삭제됩니다.4. 출결상태 정보① 출석(●) : 진행된 모든 차수의 출석체크에 출석상태인 경우② 지각(▲) : 진행된 모든 차수의 출석체크에 하나라도 출석하지 못한 경우③ 결석(X) : 진행된 모든 차수의 출석체크에 아예 출석하지 않은 경우[스마트출석 수정 및 삭제]1. 스마트출석 조회화면에서 '수정' 버튼을 클릭하면 출석 날짜와 시간을 수정할 수 있는 '출석 날짜/시간 수정' 레이어 팝업이 나타납니다.2. '출석 날짜/시간 수정' 레이어 팝업에서 날짜와 과목시간을 수정한 후 '저장' 버튼을 클릭합니다. 수정한 내용이 저장됩니다.3. 스마트출석을 삭제하려면 스마트출석 조회화면에서 '삭제' 버튼을 클릭하세요. 해당 스마트출석이 삭제됩니다.
